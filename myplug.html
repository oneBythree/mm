动态渲染table列表

使用方法

  • ①(table)表格和(nav)分页,必须用一个div包裹起来，且div的id值必须是tableList
  • ②表头thead里的th，是自定义的。根据需求自定义（操作抬头根据图标个数>=3加上class="col-xs-2"）。
  • ③循环item时根据后台接口提供的json，如item.xxx
  • ④接口数据，如果item有空值。可用三元表达式来处理item.xxx = item.xxx == ''?'---':item.xx
  • ⑤绑定函数v-on:click="xxx()"
    • 1.查看单个item : v-on:click="lookOver(item,arr1,sweetalertJson)"参数说明
      item
      循环的单个值v-for="item in tableList"对应
      arr1
      查看时弹窗框(sweetalert弹窗插件,必须引用ccs和js)列表的键值对值 [{'fieldText': '行1的文字','fieldData': item.行1的值}, {'fieldText': '行2的文字','fieldData': item.行2的值},...]
      sweetalertJson
      查看时弹窗框抬头{'swalTitle':item.抬头名称/id,'swalZh':'抬头类型'}
    • 2.修改item : v-on:click="linkUrl(Id,link)"
      Id
      修改跳转传入id（或其他唯一值）item.id
      link
      修改跳转的urlxxx.html
    • 3.删除item : v-on:click="del(item,sweetalertJson,delJson)"
      item
      循环的单个值v-for="item in tableList"对应
      sweetalertJson
      删除时弹窗框抬头{'swalTitle':item.抬头名称/id,'swalZh':'抬头类型'}
      delJson
      删除json数据 {'delUrl':'删除url','delId':item.id(删除item的id值)}
  • ⑦最后要初始化table tableInit('xxx.html');
