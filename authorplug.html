datepicker时间控件

<link rel="stylesheet" href="styles/base/daterangepicker-bs3.min.css">

<!--  datepicker.js以引入是jq插件，必须放在jq下面 -->

单日历（YYYY-MM-DD）

实例

<div class="m-icon-date ">
    <span for="date1"><i class="iconfont icon-riqi"></i></span>
    <input id="date1" bs-Datepicker type="text" class="w80" placeholder="请选择时间段" readonly>
</div>
<script type="text/javascript">
$(document).ready(function() {
    $('#date1').val(myDate.YYYYMMDD());
    $('#date1').daterangepicker({
        startDate: moment().startOf('day'),
        singleDatePicker: true
    })
})
</script>

双日历（YYYY-MM-DD hh:ss）

<div class="m-icon-date ">

    <input id="date3" bs-Datepicker type="text" class="w240" placeholder="请选择时间段" readonly>
</div>

</script>

使用方法

  • ①配置form表单data-validator-option="{theme:'bootstrap', timely:2, stopOnError:true}"
  • ②验证表单元素必须写name="xxx"(有坑),不填验证有问题
  • ③规则（rule）支持的属性：data-rule、data-rule-*、data-msg-*、data-tip、data-ok、data-target、novalidate、notimely
    • 1. data-rule 定义该字段的规则集
    • 2. data-rule-xxx 定义了一条临时规则（在data-rule中已经使用了）
    • 3.data-msg-required 更改了默认required的消息；
    • 4. data-tip 参数使得元素获得焦点时，显示友好的信息；
    • 5. data-ok 定义的消息，将在字段验证成功后显示
    • 6. data-target 如果定义，将决定消息最终显示位置。如果“#msg_holder”是字段（如input、select、textarea），消息的显示位置将转为该字段的消息位置，如果是带有“.msg-box”类的DOM，将显示在该DOM内
    • 7. novalidate 不要验证该字段，动态添加和移除该属性可以使该字段在合适的时候验证
    • 8. notimely 不要实时验证该字段，动态添加和移除该属性可以使该字段在合适的时候验证
  • ④提交表单按钮submit id不能写成submit(有坑),不能发送验证

验证规则（有需求可加）

匹配数字和字符串

整数或者小数："/^[0-9]+\.{0,1}[0-9]{0,2}$/"
只能输入数字："/^[0-9]*$/"。
只能输入n位的数字："/^\d{n}$/"。
只能输入至少n位的数字："/^\d{n,}$/"。
只能输入m~n位的数字：。"/^\d{m,n}$/"
只能输入零和非零开头的数字："/^(0|[1-9][0-9]*)$/"。
只能输入有两位小数的正实数："/^[0-9]+(.[0-9]{2})?$/"。
只能输入有1~3位小数的正实数："/^[0-9]+(.[0-9]{1,3})?$/"。
只能输入非零的正整数："/^\+?[1-9][0-9]*$/"。
只能输入非零的负整数："/^\-[1-9][]0-9"*$/"。
只能输入长度为n的字符："/^.{n}$/"。
只能输入由26个英文字母组成的字符串："/^[A-Za-z]+$/"。
只能输入由26个大写英文字母组成的字符串："/^[A-Z]+$/"。
只能输入由26个小写英文字母组成的字符串："/^[a-z]+$/"。
只能输入由数字和26个英文字母组成的字符串："/^[A-Za-z0-9]+$/"。
只能输入由数字、26个英文字母或者下划线组成的字符串："/^\w+$/"。

验证用户信息

验证用户密码："/^[a-zA-Z]\w{5,17}$/"正确格式为：以字母开头，长度在6~18之间，只能包含字符、数字和下划线。
验证是否含有/^%&',;=?$/\"等字符："/^[/^%&',;=?$\x22]+/"。
只能输入汉字："/^[\u4e00-\u9fa5]{0,}$/"
验证Email地址："/^\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$/"。
验证InternetURL："/^http://([\w-]+\.)+[\w-]+(/[\w-./?%&=]*)?$/"。
验证电话号码："/^(\(\d{3,4}-)|\d{3.4}-)?\d{7,8}$/"正确格式为："XXX-XXXXXXX"、"XXXX-XXXXXXXX"、"XXX-XXXXXXX"、"XXX-XXXXXXXX"、"XXXXXXX"和"XXXXXXXX"。
验证手机号码："/^1\d{10}$/"
验证身份证号（15位或18位数字）："/^\d{15}|\d{18}$/"。
验证一年的12个月："/^(0?[1-9]|1[0-2])$/"正确格式为："01"～"09"和"1"～"12"。
验证一个月的31天："/^((0?[1-9])|((1|2)[0-9])|30|31)$/"正确格式为；"01"～"09"和"1"～"31"。
只能输入汉字：/^[\u4e00-\u9fa5] {0,}$/
匹配双字节字符(包括汉字在内)：/^[/^\x00-\xff]/
